# SalesLearn - 页面详细说明

## 页面总览

### 员工端页面
| 页面 | 路由 | 说明 |
|---|---|---|
| 登录页 | `/login` | 员工/主管统一登录 |
| 学习首页 | `/learn` | 知识分类入口 |
| 知识卡片 | `/learn/[category]` | 卡片浏览页 |
| 测试页 | `/test/[knowledgeId]` | 做题页面 |
| 错题本 | `/review` | 错题复习列表 |
| 费曼讲解 | `/feynman/[knowledgeId]` | 录音讲解页 |
| AI 对话 | `/feynman/[knowledgeId]/chat` | 客户追问实战 |
| 个人看板 | `/dashboard` | 能力雷达图、学习进度 |

### 主管端页面
| 页面 | 路由 | 说明 |
|---|---|---|
| 管理首页 | `/admin` | 数据总览 |
| 知识库管理 | `/admin/knowledge` | 知识点 CRUD |
| 资料上传 | `/admin/knowledge/upload` | 上传切分 |
| 审核列表 | `/admin/review` | 知识点/题目审核 |
| 团队看板 | `/admin/team` | 团队数据报表 |
| 员工详情 | `/admin/team/[userId]` | 个人成长曲线 |

---

## 1. 登录页 (`/login`)

### 布局
- 居中卡片布局
- 上方：Logo + 应用名称
- 中间：手机号 + 验证码 / 账号密码登录
- 底部：角色选择（员工/主管）

### 交互
- 登录成功后根据角色跳转不同首页
- 员工 → `/learn`
- 主管 → `/admin`

---

## 2. 学习首页 (`/learn`)

### 布局
- 顶部：搜索栏 + 用户头像
- 中部：分类卡片网格（2x2）
  - 产品知识
  - 客户异议
  - 成交话术
  - 客户心理
- 底部：Tab 导航栏（学习 / 测试 / 讲解 / 我的）

### 数据展示
- 每个分类显示：总知识点数 / 已学完数 / 完成百分比

---

## 3. 知识卡片页 (`/learn/[category]`)

### 布局
- 全屏卡片，左右滑动切换
- 卡片内容：标题 → 核心要点 → 详细说明 → 案例话术
- 底部固定：进度条 + "去测试一下"按钮

### 交互
- 左滑：下一张卡片
- 右滑：上一张卡片
- 收藏：右上角星标
- 自动记录停留时长

---

## 4. 测试页 (`/test/[knowledgeId]`)

### 布局
- 顶部：进度条（当前题号/总题数）
- 中部：题干 + 4 个选项卡片
- 底部：提交按钮

### 交互流程
1. 显示题目和选项
2. 点击选项选中（高亮）
3. 点击提交 → 判定对错
4. 正确：绿色反馈 → 自动下一题
5. 错误：红色反馈 + 显示解析 → 手动下一题
6. 全部完成：显示本次得分和错题汇总

### 状态变化
- 未选：白底 + 灰色边框
- 已选：主色浅底 + 主色边框
- 正确：绿色浅底 + ✅
- 错误：红色浅底 + ❌
- 正确答案高亮：绿色边框（答错时标注正确项）

---

## 5. 错题本 (`/review`)

### 布局
- 列表布局，按复习紧急度排序
- 每项显示：知识点标题 / 上次错误日期 / 下次复习日期 / 连续正确次数
- 右侧：复习按钮 / 费曼讲解按钮

### 交互
- 点击复习 → 进入测试页
- 点击费曼 → 进入费曼讲解页
- 标签筛选：今日待复习 / 全部

---

## 6. 费曼讲解页 (`/feynman/[knowledgeId]`)

### 布局
- 顶部：知识点标题 + 阶段标识（A/B）
- 中部（阶段A）：
  - 知识点核心要点提示（折叠/展开）
  - 录音区域（居中大按钮）
  - 转写文本显示区
- 底部：提交评分按钮

### 录音交互
1. 按住录音按钮 → 开始录音（脉冲动画）
2. 松开 → 停止录音 → 语音转文字
3. 显示转写文本 → 员工可点击修正
4. 确认无误 → 点击"提交评分"
5. AI 评分中（加载动画）
6. 显示评分结果（雷达图 + 关键点列表 + 改进建议）

### 评分结果展示
- 四维雷达图
- 关键点列表：✅已讲到 / ⚠️遗漏 / ❌讲错
- 总分 + 颜色编码
- ≥80分：显示"解锁 AI 客户实战"按钮
- <80分：显示"再试一次"按钮 + 改进建议

---

## 7. AI 客户对话页 (`/feynman/[knowledgeId]/chat`)

### 布局
- 顶部：客户人设标签（小白型/比价型/懂车型）+ 轮次计数
- 中部：对话消息流（类似聊天界面）
  - AI 消息：左侧头像 + 气泡
  - 员工消息：右侧气泡
- 底部：录音按钮 + 文字输入框（二选一）

### 交互流程
1. AI 发起第一轮提问
2. 员工语音/文字回答
3. AI 继续追问或表示信服
4. 对话结束后显示评估报告

---

## 8. 个人看板 (`/dashboard`)

### 布局
- 顶部：用户信息 + 综合评分
- 中部：能力雷达图（学习完成度/识别掌握度/讲解能力）
- 下部：
  - 薄弱知识点 TOP 5
  - 近期学习趋势折线图
  - 错题本待复习数量

---

## 9. 主管 - 知识库管理 (`/admin/knowledge`)

### 布局
- 顶部：搜索栏 + 上传按钮 + 分类筛选
- 中部：知识点列表（表格布局）
  - 列：标题 / 分类 / 状态 / 关联题数 / 创建时间 / 操作
- 操作：查看 / 编辑 / 删除

---

## 10. 主管 - 团队看板 (`/admin/team`)

### 布局
- 顶部：团队整体数据（总人数/本周活跃/平均分/通过率）
- 中部：
  - 团队排行榜（综合评分排序）
  - 薄弱知识点分布（热力图/柱状图）
- 下部：
  - 员工列表（点击进入个人详情）
