# SalesLearn - 业务流程

## 1. 整体业务流程

```
主管上传资料
    ↓
AI 自动切分知识点
    ↓
主管审核（编辑/通过/打回）
    ↓
知识点发布 ──→ AI 自动生成测试题 ──→ 主管审核题目
    ↓                                      ↓
员工图文学习                          题目发布
    ↓                                      ↓
    └──────────→ 员工做测试题 ←─────────────┘
                    ↓
              答对 / 答错
              ↓         ↓
           继续      进入错题本
              ↓         ↓
              ↓    间隔复习(1/3/7/15天)
              ↓         ↓
              └────→ 费曼讲解 ←──┘
                    ↓
              阶段 A：关键点评分
                    ↓
              ≥80分？
              ↓         ↓
            是         否 → 重新练习
              ↓
        阶段 B：AI 客户追问
                    ↓
              通过评估
                    ↓
            评估看板汇总
```

## 2. 知识库管理流程

### 2.1 资料上传与切分

```
主管选择上传文件
    ↓
系统校验文件格式（PDF/Word/TXT/MD）
    ↓
调用 LLM 切分为独立知识点
    ↓
每个知识点生成结构化数据：
  - 标题
  - 核心要点（3-5 条）
  - 详细说明
  - 案例/话术示例
  - 常见误区
  - 分类标签
    ↓
进入"待审核"状态
```

### 2.2 审核工作流

```
待审核列表
    ↓
主管逐条审核
    ↓
┌──────┬──────┬──────┐
│ 通过  │ 编辑  │ 打回  │
└──┬───┴──┬───┴──┬───┘
   ↓      ↓      ↓
 发布   修改后  标记原因
 入库   再审核  返回AI重新生成
```

## 3. 学习与测试流程

### 3.1 员工学习路径

```
进入学习页面
    ↓
选择分类（产品知识/客户异议/成交话术/客户心理）
    ↓
左右滑动浏览知识卡片
    ↓
系统记录：停留时长、滑动深度
    ↓
标记"已学完" → 显示"去测试一下"按钮
    ↓
跳转对应知识点测试题
```

### 3.2 测试与错题本流程

```
显示测试题（4 选 1）
    ↓
员工作答
    ↓
┌────────┬────────┐
│  答对   │  答错   │
└──┬─────┴──┬─────┘
   ↓        ↓
 下一题   显示解析
            ↓
        加入错题本
            ↓
    设置复习间隔（1天）
            ↓
    到期提醒复习
            ↓
    ┌──────┬──────┐
    │ 答对  │ 答错  │
    └──┬───┴──┬───┘
       ↓      ↓
    streak+1  重置streak
    下次间隔   重新开始
    (3→7→15天)  (1天)
       ↓
    连续3次答对 → 移出错题本
```

## 4. 费曼讲解流程

### 4.1 阶段 A：关键点覆盖评分

```
选择知识点进入讲解
    ↓
按住录音按钮 → 语音讲解
    ↓
松开 → 语音转文字（腾讯云 ASR）
    ↓
显示转写文本 → 员工可修正
    ↓
提交评分
    ↓
AI 对比知识库标准要点
    ↓
输出评分结果：
  ✅ 讲到的关键点
  ⚠️ 遗漏的关键点
  ❌ 讲错/讲反的点
    ↓
四维评分：完整度 + 准确度 + 清晰度 + 类比恰当度
    ↓
≥80分 → 解锁阶段 B
<80分 → 显示改进建议，重新练习
```

### 4.2 阶段 B：AI 客户追问实战

```
选择客户人设（或系统随机分配）
    ↓
┌──────────┬──────────┬──────────┐
│ 小白型    │ 比价型    │ 懂车型    │
│"这是啥？" │"隔壁便宜" │ 专业挑刺   │
└──────────┴──────────┴──────────┘
    ↓
AI 发起第 1 轮提问
    ↓
员工语音/文字回答
    ↓
AI 判断是否被说服
    ↓
┌──────────┬──────────┐
│ 未说服    │ 已说服    │
│ 继续追问  │ 结束对话  │
│(最多5轮)  │          │
└──────────┴──────────┘
    ↓
生成对话评估报告
```

## 5. 评估与报告流程

```
系统自动汇总四层数据
    ↓
┌─────────────────────────────────┐
│ 学习完成度(10%) ← 模块二数据    │
│ 识别掌握度(25%) ← 模块三数据    │
│ 讲解能力(45%)   ← 模块四数据    │
│ 业绩转化(20%)   ← CRM数据(v2)  │
└─────────────────────────────────┘
    ↓
┌──────────┬──────────┐
│ 员工端    │ 主管端    │
│ 能力雷达图│ 团队排行  │
│ 薄弱提示  │ 薄弱分布  │
│          │ 成长曲线  │
└──────────┴──────────┘
```

---

## 6. 关键业务规则

| 规则 | 说明 |
|---|---|
| 费曼解锁条件 | 阶段 A 评分 ≥80 分才能进入阶段 B |
| 错题本强制讲解 | 错题本中的知识点必须走阶段 B |
| 间隔复习规则 | 答错后 1/3/7/15 天间隔复习 |
| 移出错题本条件 | 连续答对 3 次 |
| 题目审核 | AI 生成的题目必须经主管审核才能使用 |
| 知识点审核 | AI 切分的知识点必须经主管审核才能发布 |
