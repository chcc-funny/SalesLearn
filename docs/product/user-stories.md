# SalesLearn - 用户故事

## 角色定义

| 角色 | 描述 |
|---|---|
| 销售员工 | 门店一线销售，需要学习产品知识和销售话术 |
| 门店主管 | 管理销售团队，负责培训考核和业绩管理 |
| 系统管理员 | 负责系统配置和维护（MVP 阶段与主管合并） |

---

## 模块一：知识库管理

### US-101 上传培训资料
**作为** 门店主管
**我想要** 上传 PDF/Word/Markdown 等格式的培训资料
**以便** AI 能自动切分为结构化知识点，减少手动录入工作量

**验收标准**：
- [ ] 支持 PDF、Word、TXT、Markdown 格式上传
- [ ] 上传后 AI 自动切分为独立知识点
- [ ] 每个知识点包含：标题、核心要点、详细说明、案例话术、常见误区
- [ ] 切分结果进入"待审核"状态

### US-102 审核知识点
**作为** 门店主管
**我想要** 审核 AI 切分生成的知识点
**以便** 确保知识库内容准确、适用于销售培训

**验收标准**：
- [ ] 可查看待审核知识点列表
- [ ] 可编辑知识点内容
- [ ] 可通过或打回知识点
- [ ] 通过后知识点状态变为"已发布"，员工端可见

### US-103 管理知识分类
**作为** 门店主管
**我想要** 按分类管理知识点（产品知识/客户异议/成交话术/客户心理）
**以便** 员工可以按主题系统学习

## 模块二：图文学习

### US-201 浏览知识卡片
**作为** 销售员工
**我想要** 以卡片形式浏览知识点
**以便** 以直观、易消化的方式学习产品知识

**验收标准**：
- [ ] 卡片支持左右滑动浏览
- [ ] 按分类筛选知识卡片
- [ ] 支持搜索和收藏功能
- [ ] 记录学习行为（停留时长、完成标记）

### US-202 学习引导测试
**作为** 销售员工
**我想要** 学完一个知识点后能直接进入测试
**以便** 趁热打铁巩固刚学的内容

**验收标准**：
- [ ] 卡片底部显示"去测试一下"按钮
- [ ] 点击后跳转到对应知识点的测试题

## 模块三：卡片测试

### US-301 做测试题
**作为** 销售员工
**我想要** 通过做题检验自己的知识掌握程度
**以便** 找出薄弱环节并针对性提升

**验收标准**：
- [ ] 题目包含记忆题、理解题、应用题、辨析题四种题型
- [ ] 每题 4 个选项，提交后显示正确答案和解析
- [ ] 错误选项基于真实易错点，而非明显离谱的干扰项

### US-302 错题本复习
**作为** 销售员工
**我想要** 系统自动收集我的错题并按间隔提醒复习
**以便** 通过间隔复习彻底掌握薄弱知识点

**验收标准**：
- [ ] 答错的题目自动进入错题本
- [ ] 按艾宾浩斯曲线提醒复习（1天/3天/7天/15天）
- [ ] 连续答对 3 次后移出错题本
- [ ] 错题本知识点自动解锁费曼讲解任务

### US-303 审核测试题
**作为** 门店主管
**我想要** 审核 AI 自动生成的测试题
**以便** 确保题目质量和准确性

**验收标准**：
- [ ] 可查看 AI 生成的待审核题目
- [ ] 可编辑、通过或打回题目
- [ ] 可将高质量题目标记为"精品题"

## 模块四：费曼讲解

### US-401 语音讲解评分
**作为** 销售员工
**我想要** 用语音讲解知识点并获得 AI 评分
**以便** 检验自己是否真正理解并能讲清楚产品知识

**验收标准**：
- [ ] 按住录音，松开发送
- [ ] 语音自动转文字，支持查看和修正
- [ ] AI 从完整度、准确度、清晰度、类比恰当度四个维度评分
- [ ] 显示讲到的关键点、遗漏的关键点、讲错的点

### US-402 AI 客户追问实战
**作为** 销售员工
**我想要** 与 AI 模拟客户进行多轮对话练习
**以便** 提升面对不同类型客户的实战应对能力

**验收标准**：
- [ ] 阶段 A 评分 ≥80 分后解锁阶段 B
- [ ] AI 支持 3 种客户人设：小白型、比价型、懂车型
- [ ] 3-5 轮对话，员工需讲到 AI "被说服"
- [ ] 错题本知识点强制走阶段 B

## 评估看板

### US-501 个人能力看板
**作为** 销售员工
**我想要** 查看自己的学习进度和能力评估
**以便** 了解自身优势和需要提升的方向

**验收标准**：
- [ ] 显示个人能力雷达图
- [ ] 显示薄弱环节提示
- [ ] 显示学习完成度、测试分、讲解力综合评分

### US-502 团队管理看板
**作为** 门店主管
**我想要** 查看团队成员的学习和考核数据
**以便** 掌握团队整体水平，制定针对性培训策略

**验收标准**：
- [ ] 团队排行榜
- [ ] 薄弱知识点分布图
- [ ] 个人成长曲线
- [ ] 可据此决定员工是否上岗
