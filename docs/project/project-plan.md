## 4. 风险管理

| 风险 | 概率 | 影响 | 应对 |
|---|---|---|---|
| AI 出题质量不稳定 | 高 | 中 | 强制人工审核 + 精品题 few-shot |
| 语音转写准确率不够 | 中 | 中 | 支持用户手动修正 |
| LLM 成本超预算 | 中 | 低 | 分模型路由，短文本用便宜模型 |
| 员工抵触使用 | 低 | 高 | 主管引导 + 游戏化激励 |
| Prompt 调优周期长 | 高 | 中 | Phase 6 预留充足调优时间 |
